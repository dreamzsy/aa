Claude Code 接入教程

Claude Code

选择你的操作系统，按步骤完成配置，即可通过 Deep API 网关使用 Claude Code。

1安装 WSL2（Windows 子系统）
Claude Code 在 Windows 上需要通过 WSL2 运行，请先以管理员身份打开 PowerShell。
PowerShell（管理员）
PS wsl --install
# 安装完成后重启电脑，然后启动 Ubuntu
2在 WSL2 中安装 Node.js
WSL2 Ubuntu 终端
$ curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
$ sudo apt-get install -y nodejs
$ node --version
3安装 Claude Code CLI
WSL2 Ubuntu 终端
$ npm install -g @anthropic-ai/claude-code
4配置 Deep API 网关
将以下配置写入 ~/.bashrc 或 ~/.zshrc，永久生效无需每次重设。
写入环境变量
$ echo 'export ANTHROPIC_BASE_URL="https://api.deepapi.com/v1"' >> ~/.bashrc
$ echo 'export ANTHROPIC_API_KEY="sk-deep-xxxxxxxxxx"' >> ~/.bashrc
$ source ~/.bashrc
5启动 Claude Code
运行
$ claude
# 指定模型（可选）
$ claude --model claude-sonnet-4-6
